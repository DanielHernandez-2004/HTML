ESTILOS BÁSICOS DE HTML

En HTML existen infinidad de opciones para poder personalizar a nuestro gusto la base de nuestra página Web

Esta imagen estará centrada y tendrá un tamaño ajustado al centro

A continuación explicaré algunas etiquetas sencillas y cuál sería la manera de usarse:

Etiqueta STYLE dentro de esta, etiqueta podremos agregar varios parámetros para modificar la apariencia de nuestro texto como: < b> < /b>" esta etiqueta sirve para poner una parte específica del texto en NEGRITA

Etiqueta META CHARSET="UTF-8" esta etiqueta nos ayuda para que nuestro texto sea legible en cualquier navegador WEB donde este sea abierto

Etiqueta "CENTER" esta etiqueta nos ayuda para centar una parte de nuestro texto

Etiqueta "IMG =SRC" esta etiqueta nos ayuda para poder insertar una imagen en nuestro archivo de texto
